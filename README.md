# Resting Sycamore — GitHub Pages Embed Scripts

Self-contained JS embed scripts for restingsycamore.com Webflow pages.

## Usage

Each script injects styled HTML into a target div. Add to Webflow via custom code block:

```html
<div id="rsa-[page-id]"></div>
<script src="https://papiofficial.github.io/resting-sycamore-pages/[path].js"></script>
```

## Base URL

`https://papiofficial.github.io/resting-sycamore-pages/`

## Files

- `plans/` — State Medicare plan index pages (Utah, Ohio, Pennsylvania)
- `doctors/` — Doctor index pages
- `providers/` — Provider index pages  
- `specialties/` — Specialties index
- `geo/` — City, county, zip code index pages
- `questions/` — FAQ/questions index pages
- `carriers/` — Carrier index
- `plan-types/` — Plan types index
- `articles/` — Articles index

See `WEBFLOW-CHEATSHEET.md` for complete embed codes and schema JSON-LD.
